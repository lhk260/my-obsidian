# RISE: 基于组合世界模型的机器人自改进策略 阅读笔记

**论文标题**：RISE: Self-Improving Robot Policy with Compositional World Model

**核心方向**：机器人强化学习（RL），通过构建组合世界模型将机器人学习从物理环境迁移至虚拟想象空间，实现无昂贵物理交互的自改进，解决视觉 - 语言 - 动作（VLA）模型在富接触、动态操作任务中的鲁棒性不足问题。

**研究团队**：香港中文大学、Kinetix AI、香港大学、清华大学等机构联合研究

## 一、研究背景与问题提出

1. **VLA 模型的固有缺陷**：VLA 模型依托大规模预训练具备语义理解和指令跟随能力，但在富接触、动态机器人操作任务中表现脆弱，微小执行偏差会累积为失败。其核心依赖的模仿学习（IL）受限于专家演示的质量和覆盖范围，且存在**暴露偏差**—— 机器人偏离专家轨迹后缺乏恢复能力。
2. **物理世界强化学习的瓶颈**：强化学习（RL）虽能通过试错提升鲁棒性，但物理世界的在线 RL 受限于**安全风险、硬件成本、环境手动重置**，且交互为串行执行，耗时耗力，导致策略改进受限于离线数据的分布偏移。
3. **现有世界模型的不足**：现有世界模型虽试图通过模拟未来结果解决虚实差距，但存在两大问题：一是生成模型重视觉真实度、轻动作可控性，推理成本过高；二是依赖稀疏的终端奖励，缺乏对中间动作的有效学习信号，难以支撑长程操作任务。

## 二、核心贡献

1. 提出**RISE 框架**：首个将世界模型作为有效学习环境，应用于真实世界高难度机器人操作的研究，实现机器人策略的可扩展、在线自主自改进，将交互从物理环境转移至想象空间，规避物理试错的成本与风险。
2. 设计**组合世界模型**：解耦动力学预测与价值估计，为两个子模块匹配最优架构和训练目标，生成可靠的状态模拟和有信息的优势值，为策略改进提供稳定学习信号。
3. 实现在真实世界任务的显著性能提升：在动态砖块分拣、背包打包、封箱三大富接触 / 高精密任务中，相较于现有 RL/IL 基线方法，绝对性能提升 35%~45%，成功率分别达 85%、85%、95%。

## 三、核心方法：RISE 框架

RISE 的核心是**组合世界模型**+**自改进循环**，整体流程分为**策略预热**、**想象空间在线 RL**两个阶段，核心是让机器人在虚拟想象空间中完成试错和策略优化，无需物理交互。

### （一）基础公式：世界模型与 RL 形式化

1. **动力学模型**：基于多视角历史观测Ot​和动作序列at​，预测未来多视角观测o^t+1​,...,o^t+H​，公式为：
    
    o^t+1​,...,o^t+H​=D(Ot​,at​)
2. **价值模型**：为想象状态分配任务进度信号V(o^t​,ℓ)，并计算**优势值**（动作块的平均累积改进），作为策略优化的核心信号：
    
    A(ot​,at​,ℓ)=(H1​∑k=1H​V(o^t+k​,ℓ))−V(ot​,ℓ)
3. **RL 目标**：将问题建模为马尔可夫决策过程（MDP），通过优势值引导策略最大化期望回报，采用概率推理框架构建目标分布，避免直接最大化正则化目标带来的不稳定性。

### （二）核心模块 1：组合世界模型

解耦**可控动力学模型**和**进度价值模型**，各自独立优化，兼顾动作可控性、视觉真实性和失败敏感性。

1. **可控动力学模型**
    
    - 基础初始化：基于 Genie Envisioner（GE-base），继承 LTX-Video 的架构优势，实现**生成速度与质量的平衡**（比 Cosmos 快 300 倍，2 秒生成 25 帧多视角观测）。
    - 关键改进：
        
        - 加入轻量级动作编码器，在 Agibot World、Galaxea 等大规模动作标记数据集上预训练，赋予模型精细动作可控性；
        - 采用**任务中心批处理策略**：每个批次从少量任务中采样，优先保证同一场景下的动作多样性，提升任务微调效率和动作可控性；
        - 对上下文帧施加更强噪声，提升运动模糊 / 视觉伪影下的生成鲁棒性。
        
    
2. **进度价值模型**
    
    - 基础初始化：基于预训练 VLA 模型π0.5​，继承机器人中心理解能力和多视角输入适配性。
    - 双损失训练，兼顾**密集信号**和**失败敏感性**：
        
        - 进度估计损失Lprog​：学习任务的单调时间结构，提供密集的进度信号；
        - 时间差分（TD）损失LTD​：利用成功 / 失败轨迹，让模型区分执行误差与成功，提升对微小失败的敏感性；
        - 最终损失LV​=Lprog​+LTD​。
        
    

### （三）核心模块 2：策略预热（Policy Warm-up）

在自改进前，利用**离线真实世界数据**锚定策略至物理合理的行为分布，避免后续探索的盲目性：

1. 数据构成：专家演示、策略成功 / 失败轨迹、人类干预修正数据；
2. 训练策略：仅对策略轨迹分配学习到的优势值，专家 / 人类修正数据直接分配最优优势值（1），让策略吸收不同质量的动作数据；
3. 核心目标：赋予策略**优势值条件化能力**，为后续想象空间的 RL 做准备。

### （四）核心模块 3：基于世界模型的自改进循环

迭代执行**滚出阶段（Rollout）**和**训练阶段（Training）**，在想象空间完成在线 on-policy RL：

1. **滚出阶段**：
    
    - 从离线数据采样初始状态，以最优优势值（1）驱动滚出策略生成动作；
    - 动力学模型根据动作生成未来状态，价值模型计算**实际优势值**并离散化；
    - 想象状态作为下一轮滚出的输入，扩大状态覆盖范围（每次离线状态最多连续滚出 2 次，避免生成模型的误差累积）；
    - 采用指数移动平均（EMA）更新滚出策略参数。
    
2. **训练阶段**：
    
    - 以想象空间的 <观测，动作，优势值> 数据为批次，训练策略在给定优势值下生成最优动作；
    - 混合离线标记数据，避免探索过程中的**灾难性遗忘**；
    - 基于流匹配准则优化策略，让策略从高 / 低优势值的成功 / 失败中学习。
    

### （五）实现细节

1. **硬件与训练成本**：动力学模型预训练（16 张 H100，7 天）+ 任务微调（8 张 H100，3 天）；价值模型训练（8 张 H100，1 天）；策略自改进约 10k 步，批大小 64；
2. **推理开销**：组合世界模型仅用于训练，**真实世界执行时无额外计算开销**；
3. **机器人硬件**：双 7 自由度 AgileX 机器人，绝对关节控制，3 个摄像头（顶视 + 双腕部）提供多视角观测。

## 四、实验设计与结果

### （一）实验任务

选取三大真实世界高难度机器人操作任务，覆盖**动态交互、可变形物体操作、双臂精密协调**：

1. **动态砖块分拣**：从移动传送带上精准拾取彩色砖块，放入对应颜色料箱；
2. **背包打包**：打开背包、放入衣物、提起整理、拉上拉链（可变形物体操作）；
3. **封箱**：放置杯子、折叠箱盖、将卡扣卡入箱体（双臂高精密协调）。

### （二）基线方法

选取主流 IL/RL 方法，基于同一预训练 VLA 模型π0.5​实现，保证计算预算一致：

- π0.5​：仅在专家演示上微调的 IL 基线；
- π0.5​+DAgger：结合人类在线修正的交互式 IL；
- π0.5​+PPO：标准在线 RL 基线；
- π0.5​+DSRL：通过扩散隐噪声优化 VLA 的样本高效 RL；
- RECAP：基于优势值条件化的离线 RL 方法。

### （三）主要结果

RISE 在**成功率**和**阶段得分**上全面超越所有基线，关键提升：

1. 动态砖块分拣：成功率从 50%（RECAP）提升至 85%，绝对提升 35%；
2. 背包打包：成功率从 40%（RECAP）提升至 85%，绝对提升 45%；
3. 封箱：成功率从 60%（RECAP）提升至 95%，绝对提升 35%。
    
    **关键发现**：物理世界的在线 RL（PPO/DSRL）因不稳定性导致性能大幅下降，而 RISE 通过想象空间的 on-policy RL 实现稳定且显著的策略改进。

### （四）消融实验

验证 RISE 各模块 / 设计的必要性，核心结论：

1. **离线数据比例**：最优比例为 0.6，比例过低导致灾难性遗忘，过高导致过正则化（策略被限制在离线分布，无法探索更优策略）；
2. **在线动作 / 状态的作用**：同时融入在线动作和状态时性能最优，在线动作扩展动作空间探索，在线状态提供无界的训练分布；
3. **组合世界模型的模块必要性**：
    
    - 动力学模型：移除预训练导致分拣精度下降 32.15%，移除任务中心批处理导致完成率下降 30%；
    - 价值模型：移除进度损失成功率下降 20%，移除 TD 损失成功率下降 35%；
    
4. **动力学模型的可靠性**：在 PSNR、SSIM、EPE（光流端点误差）等指标上全面超越 Cosmos、原始 GE，生成结果**视觉连贯、物理一致**，无模糊 / 运动不一致问题。

### （五）附加实验

1. **优势值分箱的有效性**：高优势值分箱（Bin10）的策略性能显著高于中 / 低分箱，证明学习到的优势值有效，策略能通过优势值条件化捕捉行为多样性；
2. **训练效率**：RISE 仅用 9k 额外步骤实现 35% 的成功率提升，而 RECAP/DSRL 即使扩展 50k 步骤也无法达到相近性能，证明想象空间交互的高效性；
3. **动力学模型预训练的作用**：预训练显著提升视频生成保真度，EPE 从 1.09 降至 0.54，动作可控性大幅提升。

## 五、相关工作对比

1. **机器人学习的世界模型**：早期世界模型聚焦隐空间抽象状态建模，容量低；近期生成式世界模型重视觉真实度、轻动作可控性，且依赖稀疏终端奖励。RISE 的组合世界模型解耦动力学与价值，兼顾动作可控性和密集学习信号，且首次成功应用于真实世界高难度操作。
2. **VLA 模型的强化学习**：现有方法多在模拟器中完成 RL 微调，或在物理世界依赖离线数据重利用，牺牲策略适应性。RISE 将学习环境迁移至想象空间，实现物理世界的 on-policy RL，且通过双损失价值模型解决了现有优势值估计的过拟合和失败不敏感问题。

## 六、局限性与未来工作

1. **想象与现实的差距**：组合世界模型在罕见 / 低表征场景仍可能生成物理不合理的转换，未来需研究**不确定性感知的想象**和**物理约束的显式编码**；
2. **虚实数据平衡**：真实世界离线数据仍为必要，最优虚实数据比例需进一步调优，需深入理解离线数据的作用原理；
3. **计算成本瓶颈**：RISE 将物理交互成本转移为计算成本，训练高保真世界模型的计算开销大，未来需提升世界模型的效率；
4. **未来方向**：将学习到的模拟融入更广泛的数据生态，实现基于模型的 RL 与稀缺物理交互的互补，探索自适应、样本高效的机器人智能。

## 七、核心结论与启发

1. **核心结论**：RISE 通过**组合世界模型**构建想象空间的在线学习环境，成功将机器人 RL 从物理世界迁移至虚拟空间，实现了无昂贵物理试错的策略自改进，在真实世界富接触、高精密操作任务中大幅超越现有方法，证明了世界模型作为机器人 RL 学习环境的有效性。
2. **研究启发**：
    
    - 解耦设计是复杂世界模型的有效思路，可为不同子任务匹配最优架构和目标；
    - 针对机器人操作的**失败敏感性**设计学习信号，是提升 VLA 模型鲁棒性的关键；
    - 想象空间的 RL 是解决物理世界 RL 瓶颈的重要方向，为机器人的持续自改进提供了可行路径。
    
3. **应用价值**：RISE 有望推动机器人在工业操作、家庭服务等场景的实际应用，通过降低物理试错成本，实现机器人策略的快速迭代和鲁棒性提升，为构建自主、自适应的机器人智能奠定基础。

## 八、关键术语梳理

1. **暴露偏差（Exposure Bias）**：模仿学习中，机器人偏离专家轨迹后，因未见过相关场景而缺乏恢复能力，导致误差累积；
2. **组合世界模型（Compositional World Model）**：解耦动力学预测和价值估计的世界模型，两个子模块独立优化；
3. **任务中心批处理（Task-Centric Batching）**：优先保证同一场景下的动作多样性的批处理策略，提升动作可控性；
4. **优势值（Advantage）**：动作的实际回报与平均回报的差值，作为 RL 策略优化的核心信号；
5. **时间差分学习（Temporal-Difference Learning）**：利用相邻时刻的价值估计误差更新模型，无需等待终端奖励，适合密集信号学习。
